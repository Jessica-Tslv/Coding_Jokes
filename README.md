# Coding Jokes

> _*I came across this public API in one of the exercises on the JavaScript APIs module on the Makers Bootcamp and I used it to create a small app that fetches a random programming joke, with the purpose of adding a bit of fun to your day!*_ ☀️

## Demo

Live Demo here: https://jessica-tslv.github.io/Coding_Jokes/

## About

Coding Jokes is a tiny, interactive web app that fetches random programming jokes from the [Official Joke API](https://official-joke-api.appspot.com/) and displays them on the page. You can reveal the joke's punchline by clicking a button and you can also click to load a new joke.

I've used this project as a way to practice asynchronous functions and APIs in JavaScript.

## Technologies

- HTML
- JavaScript (ES6+, Fetch API, async/await)
